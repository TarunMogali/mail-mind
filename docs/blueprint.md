# **App Name**: MailMind

## Core Features:

- IMAP Sync: Synchronize multiple IMAP accounts in real-time using persistent connections.
- Email Storage: Store and index emails in a locally hosted Elasticsearch instance for fast searching.
- AI Categorization: Categorize emails into labels such as Interested, Meeting Booked, Not Interested, Spam, and Out of Office using AI.
- Slack Integration: Send Slack notifications for new emails categorized as 'Interested'.
- Webhook Trigger: Trigger webhooks for external automation when emails are marked as 'Interested'.
- Frontend Display: Display emails with filtering by folder/account and AI categorization.
- Suggested Replies: Use RAG to provide AI-powered suggested replies based on a product and outreach agenda, making the LLM into a reasoning tool for identifying whether that information is relevant.

## Style Guidelines:

- Primary color: Deep blue (#293B5F) to convey a sense of professionalism and trust.
- Background color: Light gray (#E7E8EA), providing a clean and neutral backdrop that won't distract from content.
- Accent color: Teal (#3D84A8) for interactive elements and important callouts to add a touch of sophistication.
- Body and headline font: 'Inter', a sans-serif font providing a modern and clean aesthetic for both headings and body text.
- Simple, professional icons for email categories and actions.
- Clean and organized layout with clear sections for email listing, filtering, and search.
- Subtle animations for loading new emails and categorization updates.
